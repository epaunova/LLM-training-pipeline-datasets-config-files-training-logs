# 🤝 Contributing to LLM Prompt Engineering

Thanks for your interest in contributing! This repository welcomes contributions related to:

### ✅ What You Can Contribute

- New prompt sets (in any domain: legal, healthcare, creative writing, etc.)
- Prompt transformation techniques (zero → few-shot, CoT, etc.)
- Prompt evaluation rubrics
- Tools to analyze or visualize prompt outputs
- Conversion scripts to/from fine-tuning formats

---

## 🧾 Guidelines

- Each prompt file should be in `.csv` or `.jsonl` format
- Please include fields like: `Prompt`, `Category`, `Tags`, `Expected Output`, `Skill Targeted`
- Make sure your prompts are clear, reproducible, and avoid hallucination triggers
- Use respectful, unbiased, inclusive examples

---

## 📦 Submitting a Pull Request

1. Fork this repo
2. Create a new branch
3. Add your files inside `/prompts/`, `/templates/`, or `/scripts/`
4. Open a pull request with a short description of your contribution
